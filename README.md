# Оценка влияния новостей эмитентов на стоимость ценных бумаг

Учебный проект (НИУ ВШЭ, 2025): NLP + машинное обучение для анализа корпоративных новостей пяти российских эмитентов и прогноза реакции рынка.

**[Полный отчёт (PDF)](docs/project_report.pdf)**

## Задача

Построить пайплайн, который:

1. Собирает корпоративные новости и котировки акций
2. Предобрабатывает тексты (токенизация, лемматизация, TF-IDF)
3. Классифицирует новости по значимости и направлению влияния на цену
4. Пробует предсказать логарифмическую доходность по тексту новостей (регрессия)

## Эмитенты

Сбербанк · Лукойл · Магнит · НЛМК · Группа Позитив — пять крупнейших секторов российского рынка (>85% капитализации).

## Результаты

### Классификация (двойная бинарная модель, Random Forest)

| Этап | F1-score |
|------|----------|
| Значимая / незначимая новость | 96% |
| Позитивная реакция (среди значимых) | 88% |
| Негативная реакция (среди значимых) | 67% |

Двойная бинарная схема (сначала отсекаем шум, затем определяем знак) лучше справилась с дисбалансом классов, чем прямая мультиклассовая классификация.

### Регрессия (логарифмическая доходность ~ TF-IDF новостей)

Random Forest и CatBoost показали сильное переобучение: высокий R² на train, отрицательный на test. Это зафиксировано в отчёте как ограничение подхода «текст → доходность» без event-study разметки.
