## Как получить данные

### 1. Корпоративные новости

```bash
python src/web_parser.py
```

Парсер собирает раскрытия с [e-disclosure.ru](https://www.e-disclosure.ru) и сохраняет Excel-файлы вида `{company}_news_last.xlsx`.

### 2. Котировки акций

Исторические цены — с Yahoo Finance, Investing.com или MOEX. В ноутбуке `01_data_merge.ipynb` ожидаются CSV с колонками даты и цены закрытия.

### 3. Размеченная выборка для классификации

Для обучения моделей тональности нужен файл с ручной разметкой (~1000 новостей):

| Поле | Значения |
|------|----------|
| `1. Существенное` | 0 — незначима, 1 — значима |
| `1. Направление` | -1 — негатив, 0 — нейтрально, 1 — позитив |
| `filtered_text` | предобработанный текст новости |

Пример структуры — в `data/sample/labeled_news_sample.csv`.

Полный файл положите в `data/labeled_news.csv` (в `.gitignore`).

### 4. Промежуточные датасеты

Ноутбуки последовательно создают:

- `all_news.csv` — объединённые новости всех эмитентов
- `merged_news.csv` — новости + лаги доходностей
- `final_df.csv` — финальный датасет для регрессии

## Старые эксперименты
